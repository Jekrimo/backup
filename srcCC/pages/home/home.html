Collect Forensically Secure Evidence from the Cloud

Cloud Catcher

Automatically collect and secure evidence from popular Cloud-based sites like Google, Facebook, Twitter, and many more.

Why Cloud Catcher?

Cloud Catcher makes it easy to automatically collect evidence from Cloud-based service providers. Provide a user name and password and Cloud Catcher will download all available user data (friends, posts, likes, emails, etc.) in a forensically secure container.

Catching the Cloud

Cloud-based data is not like traditional media sources such as computers and cell phones. Unfortunately, some service providers actively make obtaining evidence slow and cumbersome. Cloud Catcher extracts data and context such as labels, nested comments, and document revision history to help build a complete picture of an online persona.

Forensically Valid Collections

Cloud Catcher collects data using best practices for collecting and storing digital evidence from Cloud applications with the expectation that the evidence will be used in a court of law. In addition to the collected data, hash lists, verbose log files, and law enforcement standard forensically secure containers such as disk image and AFF4 are used. Collections have been designed to be easily analyzed with forensics tools such as X-Ways, EnCase, and Autopsy.

Cloud Catcher utilizes a collection target's credentials (user name and password) to login to their account and automates the collection of data via access points supplied by the Cloud application to all authenticated users. At no point is any system illegally accessed.
